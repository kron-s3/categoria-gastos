# Categoria de Gastos

Esse programa recebe dados, como nome, preço, desconto, categoria e retorna quanto você gastou em determinada categoria.

## Funcionalidades
- Classificação de gastos

## Tecnologias utilizadas
- Python

## Objetivo do projeto
- Praticar lógica de programação.

## Aprendizados
- Atribuição de x valor em uma chave em dicionáros
